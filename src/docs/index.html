Opis

Program został napisany przy użyciu framework'a Electron, który pozwala spakować aplikację webową złożoną z plików HTML, CSS, JS do aplikacji desktopowej. Do generowania wykresów została użyta biblioteka JavaScript'a Chart.js

Technologie

  • Electron JS - 22.0.0
  • Chart.js - 2.9.4
  • JavaScript
  • CSS
  • HTML 5
